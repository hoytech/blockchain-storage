Lab: Parity

Create a program that restores redundant parity backups of a file. Test it on the files below.
Your submission should be a program in any language of your choice. It should be able to run in 2 modes:
  1. Concatenation: This is the case where you have both halves of a file. This mode just glues together the two provided files.
  2. Xor: This is the case where you have one half of a file and the parity segment. This mode computes the xor of each byte in these two files to recover the other half of the file.
  • bitcoin-whitepaper-part1.txt: The first half of the backed up file
  • bitcoin-whitepaper-part2.txt: The second half of the backed up file
  • bitcoin-whitepaper.parity: The parity segment of the backed up file
